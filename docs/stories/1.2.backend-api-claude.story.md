## Story

**As a** sistema,
**I want** um servidor Fastify com TypeScript rodando, integrado ao Claude API com suporte a streaming SSE,
**so that** todos os serviços (Web UI e Bot) tenham um núcleo de IA funcional e seguro como base para as demais histórias.

## Acceptance Criteria

1. Servidor Fastify 4.x com TypeScript rodando na porta configurável via `.env` (`PORT`, padrão `3001`)
2. Endpoint `GET /health` retornando `200` com JSON `{ status: "ok", uptime: <segundos> }`
3. Endpoint `POST /api/chat/message` recebendo `{ content: string, sessionId: string }` e retornando resposta completa do Claude API
4. Endpoint `POST /api/chat/stream` com SSE streaming — resposta em chunks `data: {"type":"chunk","content":"..."}` finalizando com `data: {"type":"done"}`
5. Contexto de conversa mantido em memória por `sessionId` (rolling window de 10 mensagens — `SessionContext`)
6. Logs estruturados via Pino em todas as requisições — sem conteúdo sensível (NFR13)
7. Tratamento de erros centralizado via `AppError` + global error handler do Fastify, retornando JSON padronizado `{ error, code }`
8. Validação de variáveis de ambiente com Zod na inicialização — servidor falha com mensagem clara se variável obrigatória estiver ausente
9. Plugins registrados: `helmet`, `cors`, `rate-limit` (30 req/min por IP — NFR6)

## Risks & Mitigation

| Risk | Impact | Likelihood | Mitigation |
|------|--------|-----------|-----------|
| **Claude API quota exceeded** | Service unavailable | Low | Monitor usage; implement rate limiting |
| **Session memory leak** | Server memory grows unbounded | Medium | Implement session timeout (30 min) |
| **SSE connection drops** | Client loses stream | Low | Implement reconnect logic in frontend |
| **Env validation fails startup** | Server won't start | Low | Test with missing vars before deploy |

## Out of Scope

- ❌ **Database persistence** — Session in memory only (MVP)
- ❌ **Advanced streaming** — Basic SSE only
- ❌ **Multiple Claude models** — Sonnet only
- ❌ **Production SSL/TLS** — Development mode only

### Implementação de Referência — ChatService (streaming)

```typescript
// packages/core/src/chat/ChatService.ts
import Anthropic from '@anthropic-ai/sdk';
import { ContextStore } from './ContextStore';

export class ChatService {
  constructor(
    private claude: Anthropic,
    private contextStore: ContextStore,
  ) {}

  async *streamResponse(
    userMessage: string,
    sessionId: string
  ): AsyncGenerator<string> {
    const context = await this.contextStore.get(sessionId);

    const stream = this.claude.messages.stream({
      model: 'claude-sonnet-4-6',
      max_tokens: 4096,
      system: 'Você é ARIA, um assistente pessoal profissional. Responda sempre em português.',
      messages: [
        ...context.history.slice(-10).map(m => ({ role: m.role, content: m.content })),
        { role: 'user', content: userMessage }
      ],
    });

    let fullResponse = '';
    for await (const chunk of stream) {
      if (chunk.type === 'content_block_delta' && chunk.delta.type === 'text_delta') {
        fullResponse += chunk.delta.text;
        yield chunk.delta.text;
      }
    }

    await this.contextStore.append(sessionId, { role: 'user', content: userMessage });
    await this.contextStore.append(sessionId, { role: 'assistant', content: fullResponse });
  }
}
```

### Implementação de Referência — SSE Handler

```typescript
// apps/api/src/modules/chat/chat.controller.ts
export async function streamHandler(req: FastifyRequest, reply: FastifyReply) {
  const { content, sessionId } = req.body as ChatStreamBody;

  reply.raw.writeHead(200, {
    'Content-Type': 'text/event-stream',
    'Cache-Control': 'no-cache',
    'Connection': 'keep-alive',
  });

  try {
    for await (const chunk of chatService.streamResponse(content, sessionId)) {
      reply.raw.write(`data: ${JSON.stringify({ type: 'chunk', content: chunk })}\n\n`);
    }
    reply.raw.write(`data: ${JSON.stringify({ type: 'done' })}\n\n`);
  } catch (error) {
    reply.raw.write(`data: ${JSON.stringify({ type: 'error', code: 'AI_001' })}\n\n`);
  } finally {
    reply.raw.end();
  }
}
```

### Implementação de Referência — AppError

```typescript
// apps/api/src/shared/errors/AppError.ts
export class AppError extends Error {
  constructor(
    message: string,
    public readonly code: string,
    public readonly options?: {
      statusCode?: number;
      cause?: unknown;
      context?: Record<string, unknown>;
    }
  ) {
    super(message, { cause: options?.cause });
    this.name = 'AppError';
  }

  toJSON() {
    return {
      error: this.message,
      code: this.code,
      ...(process.env.NODE_ENV === 'development' && {
        context: this.options?.context,
      }),
    };
  }
}
```
