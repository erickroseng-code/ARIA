## Story

**As a** usuário,
**I want** consultar o status consolidado de um cliente rapidamente,
**so that** eu possa saber instantaneamente tarefas abertas, reuniões próximas, Planos criados e últimas ações sem abrir múltiplas abas.

## Acceptance Criteria

1. Comando reconhecido em variações: "status do cliente X", "resumo da Empresa X", "quanto andou X"
2. Dados consolidados retornam: tarefas abertas no ClickUp, tarefas abertas no Notion, últimas reuniões, Planos de Ataque criados
3. Resposta formatada com emojis: ✅ concluído, 🔴 atrasado, 📅 agendado, ⚠️ vencendo em 24h
4. Tarefas atrasadas aparecem primeiro, depois vencendo hoje, depois semana que vem
5. Resposta no mesmo canal onde o comando foi feito (Telegram ou Web UI)
6. Link direto para o perfil completo no Notion incluído na resposta
7. Confirmação clara: "📊 Status de *Empresa X* em [timestamp]"
8. Consulta completa em menos de 5 segundos
9. Quando cliente não encontrado: "Cliente não encontrado. Conhecidos: [lista top 3]"
10. Suporta status consolidado: histórico recente (últimas 3 ações), próximas 3 eventos

## Risks & Mitigation

| Risk | Impact | Likelihood | Mitigation |
|------|--------|-----------|-----------|
| **ClickUp/Notion timeout** | Resposta incompleta | Low | Set 3s timeout per API, fallback to partial data |
| **Client not found** | Error message | Low | Show top 3 matches, let user choose |
| **Too much data** | Message too long | Low | Paginate or summarize (top 5 tasks, 3 events) |
| **Outdated data** | Client sees stale info | Low | Cache with 5min TTL, show "last updated at" |

## Dev Notes

### Dependencies
- Depends on Story 3.2 + 3.3: Uses ClickUp + Notion task data
- Depends on Epic 2.1: Client lookup/matching
- Used by Chat flow: Called on demand when user requests status

### Architecture Reference
- Parallel API calls: ClickUp + Notion simultaneously
- Timeout: 3s per API, continue if one fails
- Caching: 5min TTL to reduce API calls
- Consolidation: Sort by priority (overdue, due today, upcoming)
- Formatting: Emoji-driven, human-readable
