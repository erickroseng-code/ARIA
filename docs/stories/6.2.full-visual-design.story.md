# Story 6.2: Full Visual Design & Design System

## Status
Done

## Story

**As a** designer,
**I want** criar um design system completo com componentes reutilizáveis,
**so that** a interface seja consistente, escalável e fácil de manter.

## Acceptance Criteria

1. Color system: primary, secondary, danger, success, warning (with variants)
2. Typography: h1-h6, body, caption, label (with Tailwind classes)
3. Spacing system: 8px base unit (gap, margin, padding tokens)
4. Component library: buttons, cards, modals, forms, tables, lists
5. Dark mode support: toggle via Tailwind dark:prefix
6. Responsive design: mobile-first approach (sm, md, lg, xl breakpoints)
7. Icon system: consistent set of SVG/icon library (20+ icons)
8. Documentation: Storybook or Notion page with component examples
9. Accessibility: WCAG 2.1 AA compliance (contrast, focus states, labels)
10. Theme switching: user can toggle light/dark mode (persisted in localStorage)
