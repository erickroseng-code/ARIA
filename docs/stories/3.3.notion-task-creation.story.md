## Story

**As a** usuário,
**I want** criar tarefas e anotações no Notion via texto ou voz,
**so that** eu possa gerenciar tarefas diretamente pelo Telegram ou Web UI usando o Notion como CRM.

## Acceptance Criteria

1. Notion API integrada com token configurável em `.env`
2. Tarefa criada em database configurável (Tasks ou similar)
3. Propriedades da tarefa: título, status ("A fazer"), data de vencimento, cliente relacionado
4. Quando cliente indicado: tarefa vinculada via relation ao perfil do cliente
5. Tarefa criada em menos de 3 segundos
6. Confirmação retorna link direto para o item no Notion
7. Formato de mensagem: "✅ Tarefa criada no Notion: *[título]* — [cliente] — [link]"
8. Rate limiting: máximo 3 requisições/segundo ao Notion (fila de retry)
9. Falha de criação: armazena em fila Redis, tenta novamente a cada 5 minutos (máx 3x)
10. Quando Notion falha: pergunta "Criar no ClickUp em vez disso?" (fallback)

## Risks & Mitigation

| Risk | Impact | Likelihood | Mitigation |
|------|--------|-----------|-----------|
| **Rate limit hit (3 req/sec)** | Task fails, goes to queue | Medium | Queue system + exponential retry |
| **Relation linking fails** | Task created but not linked to client | Low | Try link separately, alert user if fails |
| **Database schema mismatch** | Task creation fails | Low | Validate schema on startup |
| **Notion API timeout** | Task creation slow | Low | 10s timeout + queue for retry |
