## Story

**As a** usuário,
**I want** relatórios semanais e mensais com análise narrativa gerada pela IA,
**so that** eu possa entender insights acionáveis sobre meu negócio.

## Acceptance Criteria

1. Relatório semanal: produtividade, clientes atendidos, tarefas concluídas vs pendentes, destaques
2. Relatório mensal: visão consolidada, tendências, clientes mais ativos, alertas de inatividade
3. Formatação rica: títulos, tabelas, bullets, callouts
4. Análise narrativa com insights acionáveis via Claude API
5. Seção "Atenção necessária": tarefas muito atrasadas, clientes sem contato há +2 semanas
6. Gerado em menos de 60 segundos
