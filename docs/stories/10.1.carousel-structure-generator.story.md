# Story 10.1: Maverick — Carousel Structure Generator

## Status
Ready for Dev

## Story

**As a** criador de conteúdo usando o Maverick,
**I want** que o script gerado pelo Copywriter seja automaticamente transformado numa estrutura de slides de carrossel,
**so that** eu receba não só o texto do roteiro, mas já o layout pronto para criar o design.

## Acceptance Criteria

1. **Função `generateCarouselStructure(script: ScriptData): CarouselStructure`** implementada em `carousel-designer/index.ts`.
2. **Estrutura de output:** `CarouselStructure` com array de `Slide[]`, cada slide com: `position` (número), `type` ('cover' | 'content' | 'cta'), `title` (texto curto do slide), `body` (texto do slide, max 120 chars), `visual_hint` (dica visual para o designer).
3. **Lógica de quebra:**
   - Slide 1 (cover): derivado do `hook` do script — título chamativo + primeiras 120 chars do hook
   - Slides 2-N (content): `body` do script quebrado em parágrafos. Cada parágrafo vira 1 slide. Máximo de 6 slides de conteúdo.
   - Slide final (cta): derivado do `cta` do script
4. **Total de slides:** entre 3 (mínimo: cover + 1 content + cta) e 8 (máximo).
5. **`visual_cues` do script** são distribuídos nos slides de conteúdo como `visual_hint`.
6. **Exportação:** `generateCarouselStructure` e os tipos `CarouselStructure`, `Slide` exportados do módulo.

- [ ] **Task 1** — Criar `carousel-designer/index.ts` com tipos e função principal (AC: 1, 2, 3, 4, 5, 6)

  ```ts
  // aria/apps/api/src/modules/maverick/carousel-designer/index.ts

  export interface Slide {
    position: number;
    type: 'cover' | 'content' | 'cta';
    title: string;
    body: string;
    visual_hint: string;
  }

  export interface CarouselStructure {
    title: string;          // título geral do carrossel (= script.title)
    format: string;         // formato do script (ex: "Carrossel Educativo")
    total_slides: number;
    slides: Slide[];
  }

  export function generateCarouselStructure(script: {
    title: string;
    format: string;
    hook: string;
    body: string;
    visual_cues: string[];
    cta: string;
  }): CarouselStructure {
    const slides: Slide[] = [];

    // Slide 1 — Cover (do hook)
    const hookLines = script.hook.split('\n').filter(l => l.trim());
    slides.push({
      position: 1,
      type: 'cover',
      title: truncate(hookLines[0] ?? script.title, 60),
      body: truncate(hookLines.slice(1).join(' ') || script.hook, 120),
      visual_hint: script.visual_cues[0] ?? 'Imagem de impacto relacionada ao tema',
    });

    // Slides 2-N — Content (parágrafos do body)
    const paragraphs = script.body
      .split(/\n{2,}/)
      .map(p => p.trim())
      .filter(p => p.length > 10)
      .slice(0, 6);

    paragraphs.forEach((para, i) => {
      slides.push({
        position: i + 2,
        type: 'content',
        title: extractTitle(para, i + 2),
        body: truncate(para, 120),
        visual_hint: script.visual_cues[i + 1] ?? `Visual do ponto ${i + 1}`,
      });
    });

    // Slide final — CTA
    slides.push({
      position: slides.length + 1,
      type: 'cta',
      title: 'Próximo passo',
      body: truncate(script.cta, 120),
      visual_hint: 'Call-to-action claro, fundo contrastante',
    });

    return {
      title: script.title,
      format: script.format,
      total_slides: slides.length,
      slides,
    };
  }

  function truncate(text: string, maxLen: number): string {
    if (text.length <= maxLen) return text;
    return text.slice(0, maxLen - 3).trimEnd() + '...';
  }

  function extractTitle(paragraph: string, slideNum: number): string {
    // Tenta usar a primeira frase curta como título, senão gera um genérico
    const firstSentence = paragraph.split(/[.!?]/)[0]?.trim() ?? '';
    if (firstSentence.length > 5 && firstSentence.length <= 50) {
      return firstSentence;
    }
    return `Ponto ${slideNum - 1}`;
  }
  ```
