## Story

**As a** usuário,
**I want** receber notificações proativas sobre minhas reuniões e tarefas,
**so that** eu nunca esqueço compromissos importantes e prazos.

## Acceptance Criteria

1. Calendar reminder: 15 minutes before meeting → notification "📅 Reunião em 15 min"
2. Task overdue: when task becomes overdue → "🔴 Tarefa atrasou!" (ClickUp + Notion)
3. Notification delivery: Telegram (immediate) + Web UI banner (if user online)
4. Smart context: show brief task/meeting summary if available
5. Mutable notifications: user can snooze (15 min, 1 hour, 1 day) or dismiss
6. Delivery channels: Telegram by default; Web UI if user has tab open
7. Rate limiting: max 1 notification per task/meeting (avoid spam)
8. Timezone aware: 15-min window calculated in user's timezone
9. Extensible: add email/Slack notifications in Phase 2
10. Error handling: failed notification → retry queue; user sees missed notification on login

## Risks & Mitigation

| Risk | Impact | Likelihood | Mitigation |
|------|--------|-----------|-----------|
| **WebSocket connection drops** | Web notifications miss | Medium | Fallback to polling; auto-reconnect with exponential backoff |
| **Notification spam** | User annoyed; ignores notifications | Medium | Deduplication: 1 notification per event; snooze logic |
| **Timezone calculation wrong** | Notification at wrong time | Low | Validate timezone against Intl API; test DST |
| **Telegram delivery fails** | User doesn't get notification | Low | Queue system; retry with exponential backoff |
