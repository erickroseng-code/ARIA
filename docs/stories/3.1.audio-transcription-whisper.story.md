## Story

**As a** usuário,
**I want** enviar mensagens de voz pelo Telegram ou gravar áudio na Web UI e ter o conteúdo transcrito automaticamente,
**so that** eu possa criar tarefas, agendar reuniões e interagir com a ARIA sem precisar digitar — apenas falando naturalmente.

## Acceptance Criteria

1. Integração com OpenAI Whisper API (`whisper-1`) configurada via `OPENAI_API_KEY` no `.env`
2. Telegram: mensagem de voz (`.ogg`) recebida → transcrita → exibida + processada como comando de texto
3. Web UI: botão de gravação no `InputBar` → grava → transcreve → exibe transcrição → processa como mensagem
4. Indicador visual enviado antes da transcrição: `"🎙 Transcrevendo áudio..."` (Telegram) / OrbState `listening` → `processing` (Web UI)
5. Transcrição exibida ao usuário antes de processar: `"🎙 *Você disse:* agenda reunião com João amanhã às 14h"`
6. Áudio deletado da memória imediatamente após transcrição — nunca gravado em disco (NFR11)
7. Áudios com mais de **5 minutos** rejeitados antes do envio ao Whisper: `"⏱ Áudio longo demais (6:32). Limite: 5 minutos."`
8. Endpoint `POST /api/audio/transcribe` retorna `{ text: string; durationSec: number }` com auth JWT
9. Endpoint `POST /api/audio/process` transcreve + processa como comando → SSE com resposta da ARIA

## Dev Notes

### OpenAI Whisper API — Referência

```typescript
// packages/integrations/src/audio/WhisperClient.ts
import OpenAI from 'openai';

export class WhisperClient {
  private client: OpenAI;

  constructor(apiKey: string) {
    this.client = new OpenAI({ apiKey });
  }

  async transcribe(
    buffer: Buffer,
    filename: string,
    mimeType: string
  ): Promise<TranscriptionResult> {
    const file = new File([buffer], filename, { type: mimeType });

    const response = await this.client.audio.transcriptions.create({
      file,
      model: 'whisper-1',
      language: 'pt',
      response_format: 'verbose_json',  // inclui 'duration'
    });

    // response.duration é em segundos (float)
    if (response.duration && response.duration > 300) {
      const minutes = Math.floor(response.duration / 60);
      const seconds = Math.round(response.duration % 60);
      throw new AppError('AUDIO_001', `Áudio muito longo (${minutes}:${String(seconds).padStart(2,'0')}). Limite: 5 minutos.`);
    }

    return {
      text: response.text,
      durationSec: response.duration ?? undefined,
      language: response.language ?? 'pt',
    };
  }
}
```

**Nota:** A validação de duração do Telegram (`ctx.message.voice.duration`) deve ocorrer ANTES de chamar o Whisper para economizar chamadas de API. O Whisper também valida, mas como fallback.

### Telegram Voice Download — Referência

```typescript
// apps/bot/src/handlers/voice.handler.ts
async function handleVoice(ctx: MyContext) {
  const voice = ctx.message?.voice;
  if (!voice) return;

  // Validação prévia de duração (sem chamar API)
  if (voice.duration > 300) {
    const m = Math.floor(voice.duration / 60);
    const s = voice.duration % 60;
    await ctx.reply(`⏱ Áudio longo demais (${m}:${String(s).padStart(2,'0')}). Limite: 5 minutos.`);
    return;
  }

  const interim = await ctx.reply('🎙 Transcrevendo áudio...');

  // Download em memória
  const file = await ctx.getFile();
  const fileUrl = `https://api.telegram.org/file/bot${BOT_TOKEN}/${file.file_path}`;
  const response = await fetch(fileUrl);
  const arrayBuffer = await response.arrayBuffer();
  const buffer = Buffer.from(arrayBuffer);  // Nunca escrito em disco — NFR11

  // Processar via API
  const result = await apiClient.post('/api/audio/process', {
    buffer, filename: 'voice.ogg', mimeType: 'audio/ogg',
    sessionId: `tg_${ctx.chat.id}`
  });

  // Editar mensagem interim com transcrição
  await ctx.api.editMessageText(
    ctx.chat.id, interim.message_id,
    `🎙 *Você disse:* ${escapeMarkdownV2(result.transcript)}`,
    { parse_mode: 'MarkdownV2' }
  );
  // Enviar resposta da ARIA
  await ctx.reply(result.ariaResponse, { parse_mode: 'MarkdownV2' });
}
```

### Web UI — MediaRecorder + `useAudioInput`

```typescript
// apps/web/src/hooks/useAudioInput.ts
export function useAudioInput() {
  const [isRecording, setIsRecording] = useState(false);
  const [isTranscribing, setIsTranscribing] = useState(false);
  const [transcript, setTranscript] = useState<string | null>(null);
  const [error, setError] = useState<string | null>(null);
  const mediaRecorderRef = useRef<MediaRecorder | null>(null);
  const chunksRef = useRef<Blob[]>([]);
  const { setOrbState } = useUIStore();

  const startRecording = async () => {
    try {
      const stream = await navigator.mediaDevices.getUserMedia({ audio: true });
      const recorder = new MediaRecorder(stream, { mimeType: 'audio/webm; codecs=opus' });
      chunksRef.current = [];
      recorder.ondataavailable = (e) => chunksRef.current.push(e.data);
      recorder.onstop = async () => {
        const blob = new Blob(chunksRef.current, { type: 'audio/webm' });
        stream.getTracks().forEach(t => t.stop()); // liberar microfone
        setIsRecording(false);
        setIsTranscribing(true);
        setOrbState('processing');
        try {
          const { text } = await transcribeAudio(blob);
          setTranscript(text);
        } catch (err) {
          setError('Falha ao transcrever. Tente novamente.');
        } finally {
          setIsTranscribing(false);
          setOrbState('idle');
        }
      };
      mediaRecorderRef.current = recorder;
      recorder.start();
      setIsRecording(true);
      setOrbState('listening');
    } catch {
      setError('Permissão de microfone negada. Verifique as configurações do navegador.');
    }
  };

  const stopRecording = () => mediaRecorderRef.current?.stop();

  return { isRecording, isTranscribing, transcript, error, startRecording, stopRecording,
           clearTranscript: () => setTranscript(null) };
}
```

### Erro Codes de Áudio

| Código | Trigger | Mensagem ao Usuário |
|--------|---------|---------------------|
| `AUDIO_001` | Duração > 5 min | `"⏱ Áudio longo demais. Limite: 5 minutos."` |
| `AUDIO_002` | MIME type inválido | `"❌ Formato de áudio não suportado. Use .ogg, .mp3 ou .m4a"` |
| `AUDIO_003` | Falha Whisper API | `"⚠️ Não consegui transcrever o áudio. Tente novamente ou envie como texto."` |
