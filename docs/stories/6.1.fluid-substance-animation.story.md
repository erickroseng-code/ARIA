## Story

**As a** usuário,
**I want** ver o assistente como substância fluida animada,
**so that** a interface seja mais imersiva e responsiva aos estados da conversa.

## Acceptance Criteria

1. Substância fluida renderizada com Three.js/WebGL em WebGLRenderer
2. 6 estados de animação: IDLE, LISTENING, PROCESSING, RESPONDING, SUCCESS, ERROR
3. IDLE: pulsação suave, movimentação orgânica (2-3s por ciclo)
4. LISTENING: ondas aceleradas, deformação em resposta a input de áudio
5. PROCESSING: rotação + deformação dinâmica, velocidade aumentada
6. RESPONDING: animação suave tipo "respiração", resposta a mensagens
7. SUCCESS: brilho gradual, expansão final, transição suave para IDLE
8. ERROR: vibração rápida (0.3s), cor vermelha temporária, volta ao IDLE
9. Transições entre estados suaves (0.5s interpolação)
10. Performance: 60 FPS em desktop, 30 FPS em mobile (fallback geometry)
11. Canvas responsivo: redimensiona com viewport
12. Suporta fallback 2D (Canvas 2D) para navegadores sem WebGL

## Risks & Mitigation

| Risk | Impact | Likelihood | Mitigation |
|------|--------|-----------|-----------|
| **WebGL not supported** | Interface breaks on some devices | Medium | Implement Canvas 2D fallback |
| **Performance degradation** | Frame rate drops below 30 FPS | Medium | Use geometry instancing; optimize shader |
| **Shader compilation errors** | Visual artifacts, inconsistent behavior | Low | Test on multiple GPUs; provide pre-compiled fallbacks |
| **Audio input lag** | LISTENING state appears delayed | Low | Buffer audio frames; use requestAnimationFrame sync |
| **State transition janky** | Poor user experience | Low | Implement Bezier easing; test all transitions |

## Out of Scope

- ❌ **Custom shader language** — GLSL only
- ❌ **Physics simulation** — Pre-calculated animation curves
- ❌ **Particle effects** — Solid geometry only
- ❌ **VR/AR support** — Desktop/mobile web only

## Dev Notes

Uses Three.js with custom shader for fluid-like geometry. Audio reactive via Web Audio API FFT. Mobile fallback to simpler geometry and Canvas 2D. State machine manages all 6 states with smooth Bezier transitions. Performance targets: 60 FPS desktop, 30 FPS mobile (tested on iPhone 12+).
