# Story 5.4: On-Demand Document Analysis & Insights

## Status
Done

## Story

**As a** usuário,
**I want** analisar documentos carregados para extrair insights e recomendações,
**so that** eu possa extrair valor de documentos sem ler tudo manualmente.

## Acceptance Criteria

1. File upload support: PDF, DOCX, TXT, images (OCR Phase 2)
2. File size limit: max 10MB per document
3. Parsing: extract text + structure (headings, lists, tables)
4. Claude analysis: summarize + identify key points + extract recommendations
5. Output: summary, key points, recommendations, sentiment analysis
6. Analysis completed within 1 minute for typical documents
7. Results stored in Notion as document page
8. User notified when analysis complete (Telegram + Web UI)
9. Support batch analysis (multiple documents at once)
10. Error handling: file parsing fails → ask user to retry with PDF
