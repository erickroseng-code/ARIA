## Story

**As a** sistema,
**I want** coletar dados do Notion, ClickUp e Google Calendar num modelo estruturado,
**so that** relatórios podem ser gerados com dados consolidados.

## Acceptance Criteria

1. Coleta do ClickUp: tarefas concluídas, pendentes, atrasadas, criadas no período
2. Coleta do Notion: clientes ativos, Planos criados, reuniões registradas
3. Coleta do Google Calendar: reuniões realizadas e agendadas
4. Dados normalizados em modelo JSON estruturado
5. Erro em uma fonte não bloqueia o relatório — marcada como "dados parciais"
6. Modelo extensível para novas fontes
7. Coleta completa em menos de 10 segundos

## Complexity Estimate

**T-Shirt Sizing:** Medium (M)
**Story Points:** 8-10 points
**Estimated Dev Time:** 3 days
**Estimated QA Time:** 1-2 days

**Complexity Drivers:**
- Multiple API calls in parallel
- Data normalization
- Error handling and graceful degradation
- Caching strategy

## Risks & Mitigation

| Risk | Impact | Likelihood | Mitigation |
|------|--------|-----------|-----------|
| **API rate limits** | Aggregation fails | Medium | Cache results for 1h; batch requests |
| **One source fails** | Incomplete report | Low | Graceful degradation; mark as partial |
| **Slow aggregation** | Exceeds timeout | Low | Parallel fetching; set timeout for each API |

## Out of Scope

- ❌ **Real-time aggregation** — Scheduled only
- ❌ **Historical data** — Current state only
- ❌ **Custom metrics** — Fixed set for MVP
