## Story

**As a** usuário,
**I want** criar e consultar eventos no Google Calendar via comando,
**so that** eu possa gerenciar minha agenda integralmente dentro da ARIA.

## Acceptance Criteria

1. Integração com Google Calendar API via OAuth2 (User consent flow)
2. Criação de evento: "agenda reunião com Empresa X na quinta às 14h"
3. Consulta: "o que tenho hoje?", "quais reuniões tenho essa semana?"
4. Evento criado com: título, data, hora, duração padrão 1h (ajustável via parâmetro)
5. Confirmação: "📅 Reunião com *Empresa X* agendada para quinta, 14h" com link direto
6. Cancelamento por comando: "cancela reunião com X"
7. Suporte a zona de horário PT-BR automaticamente

## Risks & Mitigation

| Risk | Impact | Likelihood | Mitigation |
|------|--------|-----------|-----------|
| **OAuth token expiration** | User loses calendar access | High | Implement refresh token rotation; check token validity on each call |
| **User not authenticated with Google** | Can't create calendar event | Medium | Clear error message with link to Google account setup |
| **Timezone conflicts** | Wrong time displayed | Medium | Use user's timezone from account settings; test DST transitions |
| **Calendar API rate limits** | Requests blocked | Low | Implement caching for 5 min; batch requests |

## Out of Scope

- ❌ **Calendar sharing** — Single user only
- ❌ **Invite attendees** — Create event for user only
- ❌ **Recurring events** — One-off events only
- ❌ **Video meeting links** — Manual Google Meet setup
- ❌ **Multiple calendars** — Primary calendar only
