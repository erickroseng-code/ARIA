## 2. Notas de UX

1.  **Hierarquia Visual:** O "Raio-X" deve ser o elemento mais urgente. Vermelho/Amarelo/Verde para status instantâneo.
2.  **Redução de Carga Cognitiva:** Apenas 3 opções de caminhos. Não 10, não 50. O Alex sofre de paralisia de escolha; o Maverick cura isso limitando as opções às *melhores*.
3.  **Explicação do "Porquê":** O Alex é um expert. Ele não vai seguir uma IA cegamente. O Maverick precisa *vender* a ideia mostrando os dados ("Por que isso vai funcionar?").
4.  **Tom de Voz da Interface:** Confidante, direto, parceiro estratégico. (Ex: "Sua audiência está cansada", não "Parece que o engajamento caiu").
