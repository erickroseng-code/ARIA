# Task: Detect Brand Voice

## Context
Executada pelo **Maverick Scout**. O objetivo é destilar o tom de voz e a personalidade da marca a partir dos dados extraídos do perfil social.

## Input
- `raw_profile_data`: Dados extraídos de bio e posts.

## Steps
1. **Análise de Vocabulário:** Liste as palavras e expressões mais usadas pelo cliente.
2. **Personalidade da Marca:** Classifique o tom (ex: Informativo, Provocador, Compassivo).
3. **Padrões de Comunicação:**
   - Como o cliente saúda o público?
   - Ele usa gírias ou termos técnicos?
   - Como ele termina os posts (CTA)?
4. **Resumo Executivo:** Gere um "Guia de Voz" de uma página para o **Maverick Copywriter**.

## Output
- `brand_voice_guide`: Documento com as diretrizes de voz da marca.
- `key_adjectives`: Lista de 3-5 adjetivos que definem a marca.
