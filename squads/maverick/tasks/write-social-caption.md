# Task: Write Social Caption

## Context
Executada pelo **Maverick Copywriter**. Focada em transformar ideias em textos de legenda persuasivos para redes sociais.

## Input
- `topic`: O tema do post.
- `framework`: Modelo de copy a usar (ex: AIDA, PAS).
- `brand_voice_guide`: Diretrizes de tom de voz.

## Steps
1. **Redação do Gancho:** Crie uma primeira frase que pare o scroll do usuário.
2. **Desenvolvimento:** Aplique o framework de copy escolhido.
3. **Chamada para Ação:** Adicione um CTA claro e direto.
4. **Otimização de SEO:** Adicione hashtags e palavras-chave relevantes.
5. **Formatação:** Use emojis e quebras de linha para facilitar a leitura.

## Output
- `caption_text`: Texto da legenda finalizado.
