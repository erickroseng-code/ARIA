# Task: Write Creative Script (Roteirização)

## Contexto
O Plano de Ação foi aprovado pelo humano. O Strategist definiu o tópico e o ângulo. Agora, o **Copywriter** precisa transformar "ideia" em "roteiro executável".

## Objetivo
Gerar roteiros completos, prontos para gravação (vídeo) ou design (carrossel), seguindo estritamente o tom de voz identificado pelo Scout e a estrutura técnica da plataforma.

## Passos de Execução

1.  **Seleção de Modelo (Template):**
    *   Identificar o formato do conteúdo (Reels, Stories, Carrossel).
    *   Selecionar o template narrativo adequado da base (ex: AIDA, PAS, Storytelling de Herói).

2.  **Escrita do Roteiro (Drafting):**
    *   **Gancho (Hook):** Os primeiros 3 segundos (vídeo) ou slide 1 (carrossel). Deve ser magnético.
    *   **Corpo (Body):** Desenvolvimento do conteúdo com densidade informacional.
    *   **CTA (Call to Action):** O que a pessoa deve fazer depois.

3.  **Refinamento de Tom (Tone Check):**
    *   Aplicar o "Vocabulary Filter" do perfil (palavras que ele usa vs palavras que ele nunca usaria).
    *   Garantir que a linguagem não soe robótica.

## Output Esperado

### Se for Vídeo (Reels/TikTok):
```markdown
**Título:** [Nome do Vídeo]
**Formato:** Reels (Falado para câmera)

| Tempo | Visual (Sugestão) | Áudio (Falar) |
| :--- | :--- | :--- |
| 00:00-00:03 | Close no rosto, expressão séria | "Pare de errar nisso se você quer crescer." |
| 00:03-00:15 | Corta para tela mostrando gráfico | "A maioria ignora o CAC, mas é ele que..." |
| ... | ... | ... |
| 00:50-01:00 | Volta para rosto | "Comenta 'AULA' se quer a parte 2." |
```

### Se for Carrossel:
```markdown
**Título:** [Nome do Carrossel]
**Formato:** Carrossel (7 slides)

**Slide 1 (Capa):**
- Título Grande: "O Fim do Marketing?"
- Subtítulo: "O que ninguém te contou."
- Visual: Fundo preto, texto branco.

**Slide 2:**
- Texto: "Você acha que marketing é só postar..."
- Visual: Ícone de alerta.

...

**Slide 7 (CTA):**
- Texto: "Salve para consultar depois."
```
