# Task: Apply Tone to Text

## Context
Executada pelo **Maverick Copywriter**. Serve para reescrever ou ajustar textos existentes para que soem exatamente como a marca do cliente.

## Input
- `source_text`: O texto original a ser ajustado.
- `brand_voice_guide`: Guia de tom de voz gerado pelo Scout.

## Steps
1. **Análise de Diferença:** Compare o tom do `source_text` com o `brand_voice_guide`.
2. **Substituição de Vocabulário:** Troque palavras e expressões para alinhar com o dicionário da marca.
3. **Ajuste de Estrutura:** Altere o ritmo das frases (mais curtas para tons casuais, mais elaboradas para formais).
4. **Validação Final:** Verifique se a "alma" da marca está presente no texto final.

## Output
- `adjusted_text`: O texto com o tom de voz aplicado corretamente.
