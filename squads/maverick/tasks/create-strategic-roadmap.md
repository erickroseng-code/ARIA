# Task: Create Strategic Roadmap

## Context
Executada pelo **Maverick Strategist**. Organiza as sugestões em um cronograma estratégico de publicações.

## Input
- `content_suggestions`: Lista de pautas sugeridas.
- `frequency`: Frequência de postagens (ex: 3x por semana).

## Steps
1. **Sequenciamento:** Organize as pautas em uma ordem lógica de consciência (Aprendizado -> Problema -> Solução -> Oferta).
2. **Definição de Formatos:** Atribua um formato para cada dia (ex: Segunda - Carrossel Educativo, Quarta - Reels de Autoridade, Sexta - Post de Venda).
3. **Calendário de Execução:** Crie um cronograma de 7, 15 ou 30 dias.
4. **Resumo de Metas:** Defina o que o cliente deve esperar de resultado desse roadmap.

## Output
- `strategic_roadmap`: Documento com o calendário de publicações.
- `roadmap_path`: Caminho para o arquivo salvo no diretório de snapshots.
