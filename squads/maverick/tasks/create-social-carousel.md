# Task: Create Social Carousel

## Context
Executada pelo **Maverick Designer**. Organiza e cria as múltiplas telas de um carrossel para Instagram ou LinkedIn, garantindo fluidez visual entre os slides.

## Input
- `carousel_script`: O roteiro dividido por telas (Capa, Slides, CTA).
- `design_style`: Estilo visual em `./data/references/`.
- `figma_config`: ID do arquivo ou frame base no Figma.

## Steps
1. **Decomposição Visual:** Analise cada tela do carrossel e defina o foco visual (Texto vs. Imagem).
2. **Criação de Slides:** No Figma, duplique ou crie frames em sequência para cada slide.
3. **Design de Continuidade:** Garanta que elementos visuais "vazem" de um slide para o outro, incentivando o deslizar.
4. **Inserção de Texto:** Posicione os ganchos e blocos de texto conforme o roteiro do Copywriter.
5. **Revisão Visual:** Verifique o alinhamento, legibilidade e hierarquia visual das fontes.

## Output
- `figma_carousel_link`: URL para o carrossel no Figma.
- `carousel_preview`: Imagens de pré-visualização (se exportáveis).

## Validation
- O carrossel tem uma capa que chama a atenção?
- A leitura é fluida entre os slides?
- O CTA final é visível e claro?
