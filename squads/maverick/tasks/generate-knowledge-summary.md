# Task: Generate Knowledge Summary

## Context
Executada pelo **Maverick Scholar**. Cria resumos executivos de conceitos complexos para apoiar o planejamento estratégico.

## Input
- `concept_name`: O conceito ou autor a ser resumido.
- `context`: Onde esse resumo será usado (ex: post de Instagram, roteiro de vídeo).

## Steps
1. **Recuperação:** Busque na base de conhecimento tudo o que diz respeito ao `concept_name`.
2. **Sintetização:** Crie uma explicação de "nível especialista" mas acessível.
3. **Exemplificação:** Extraia pelo menos um exemplo prático contido nos livros da base.
4. **Citação:** Formate as referências (Livro, Autor) de acordo com o padrão do Maverick.

## Output
- `concept_summary`: Resumo consolidado e pronto para uso estratégico.
